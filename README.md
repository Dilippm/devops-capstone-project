# devops-capstone-project

![Build Status](https://github.com/Dilippm/devops-capstone-project/actions/workflows/ci-build.yaml/badge.svg)

This project is part of the DevOps Capstone Project.  
It implements a Customer Account Microservice that provides REST API endpoints to create, read, update, delete, and list customer accounts.

The project demonstrates Agile planning, CI/CD, containerization, and Kubernetes deployment.
